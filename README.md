## Online-retail-sql-project
A simple SQL project designed to analyze an online retail store’s data using SQL queries. This project includes tables such as Customers, Products, Orders, and Order_Items, along with multiple queries that help understand sales trends, customer behavior, and product performance.

## Overview
This project contains a complete SQL case study with schema design, realistic data generation, and analysis queries.

## Files Included
- schema.sql – Database schema  
- insert_data.sql – Contains 300+ rows  
- queries.sql – Analytical SQL queries  

## Highlights
- Customer analytics  
- Product & revenue insights  
- Monthly trends  
- Window functions
